## Usage

```
npm link    # one-time setup
sif         # start a cycle
```

**Controls:**
- `spacebar` — skip forward to next zone
- `q` — quit early
- `Ctrl+C` — exit
- Any key dismisses the finish screen

## Zones

| Zone  | Time       | Description                                             |
|-------|------------|---------------------------------------------------------|
| GATHER| 00:00–14:59| context assembly — client notes — screenshots           |
| BUFFER| 15:00–16:59| problem write-up — similar issues — historical insights |
| WORK  | 17:00–21:59| work — RESPOND \|\| RESOLVE                             |
| CLOSE | 22:00–25:00| admin matrix — DO \|\| DECIDE \|\| DELEGATE \|\| DELETE |

## Design

- Zero dependencies — raw ANSI escape codes, Node.js only
- Solarized Dark color palette (set terminal background to `#002b36`)
- No sound, no system notifications — visual only
- No pause — only forward. Time doesn't wait.

## Customization

- **Zone names, timing, descriptions** — edit `src/zones.js`
- **UI chrome** (labels, status text, finish screen copy) — edit the `LABELS` object at the top of `src/renderer.js`
- **Terminal background** — set to `#002b36` for proper Solarized Dark rendering

## Architecture

```
bin/sif.js        CLI entry, keypress handling, tick loop
src/zones.js      Zone definitions and timing config
src/timer.js      Timer state, elapsed tracking, zone skipping
src/renderer.js   LABELS config, terminal UI, progress bar, glitch finish screen
```
